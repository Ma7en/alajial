Lesson Plan (خطة الدرس)
Lesson Title (عنوان الدرس):
Types of Files

Class (الفصل):
Second Grade

Day (اليوم):
Thursday

Lesson Objectives (أهداف الدرس):
By the end of this lesson, students will be able to:
1. Understand different types of files (e.g., documents, images, videos).
2. Identify file extensions (e.g., .docx, .jpg, .mp4).

Period (الفترة الزمنية):
45 minutes

Date (التاريخ):
[Insert Date]

Strategy (الاستراتيجية):
Interactive Lecture, Hands-on Activity

New Vocabularies (المفردات الجديدة):
1. File types
2. Document
3. Image
4. Video
5. File extension

Teaching Aids (الوسائل التعليمية):
1. Computer Lab
2. Computers
3. Examples of different file types

Time (الوقت):
- Warm-up: 5 minutes
- Procedures: 30 minutes
- Evaluation: 5 minutes
- Homework: 5 minutes

Warm-up (التهيئة):
- Greet the students and ask if they have used different types of files before.

Procedures (الإجراءات):
1. Introduction (5 minutes):
    - Introduce the concept of different file types and their importance.
2. Presentation (15 minutes):
    - Show examples of different file types (documents, images, videos).
    - Explain file extensions and their uses.
3. Hands-on Activity (10 minutes):
    - Let students explore and identify different file types on the computer.

Evaluation (التقييم):
- Ask students questions about different file types and extensions.
- Observe their ability to identify file types.

Homework (الواجب المنزلي):
- Create a list of different file types and their extensions.

خطة الدرس (باللغة العربية)
عنوان الدرس:
أنواع الملفات

الفصل:
الصف الثاني الابتدائي

اليوم:
الخميس

أهداف الدرس:
في نهاية هذا الدرس، سيتمكن الطلاب من:
1. فهم أنواع الملفات المختلفة (مثل المستندات والصور ومقاطع الفيديو).
2. التعرف على امتدادات الملفات (مثل .docx و.jpg و.mp4).

الفترة الزمنية:
45 دقيقة

التاريخ:
[تاريخ اليوم]

الاستراتيجية:
محاضرة تفاعلية، نشاط عملي

المفردات الجديدة:
1. أنواع الملفات
2. مستند
3. صورة
4. فيديو
5. امتداد الملف

الوسائل التعليمية:
1. معمل الحاسب الآلي
2. أجهزة الحاسب الآلي
3. أمثلة على أنواع الملفات المختلفة

الوقت:
- التهيئة: 5 دقائق
- الإجراءات: 30 دقيقة
- التقييم: 5 دقائق
- الواجب المنزلي: 5 دقائق

التهيئة:
- تحية الطلاب وسؤالهم إذا كانوا قد استخدموا أنواعًا مختلفة من الملفات من قبل.

الإجراءات:
1. المقدمة (5 دقائق):
    - تقديم مفهوم أنواع الملفات المختلفة وأهميتها.
2. العرض (15 دقيقة):
    - عرض أمثلة على أنواع الملفات المختلفة (المستندات والصور ومقاطع الفيديو).
    - شرح امتدادات الملفات واستخداماتها.
3. النشاط العملي (10 دقائق):
    - دع الطلاب يستكشفون ويتعرفون على أنواع الملفات المختلفة على الحاسب الآلي.

التقييم:
- سؤال الطلاب عن أنواع الملفات وامتداداتها.
- مراقبة قدرتهم على التعرف على أنواع الملفات.

الواجب المنزلي:
- إنشاء قائمة بأنواع الملفات وامتداداتها.
